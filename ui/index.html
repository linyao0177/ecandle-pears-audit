How this works

Every payment generates an ed25519 signature on the device, appended to a Hypercore append-only feed. This audit page replicates the feed locally and verifies every signature against the device's ed25519 public key. The cryptographic chain holds even if every eCandle / Tether / Arkreen server goes dark.
